Welcome to Serenia

Serenia is a conceptual smart city designed as part of the DAA course project. The goal is to represent real-world systems using algorithmic principles such as Graph Traversal, Shortest Path Algorithms, Minimum Spanning Trees, Greedy Methods, and Dynamic Programming. This project demonstrates how theoretical algorithm design translates into practical city-level planning and optimization.
